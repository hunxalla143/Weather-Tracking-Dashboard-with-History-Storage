# Weather Tracking Dashboard

A professional Python application that tracks real-time weather data using OpenWeatherMap API and stores historical weather records for analysis.

## Features

- Live weather tracking
- Multi-city monitoring
- Historical weather storage
- CSV export
- JSON export
- Temperature trend graphs
- Weather alerts
- Auto refresh mode
- City filtering
- Statistics dashboard
- Modular architecture

## Technologies

- Python
- OpenWeatherMap API
- Pandas
- Matplotlib
- Requests
- Tabulate
